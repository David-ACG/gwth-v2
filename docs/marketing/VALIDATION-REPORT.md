## Issues Found and Fixed

### Round 1 — Student Perspective Review (see REVIEW-NOTES-round1.md)

Key changes made to the main document:
1. Moved empowerment messaging earlier — less doom before offering hope
2. Added mid-career professional advantages section
3. Strengthened "no coding" as headline-level message
4. Added "5 hours a week" prominence
5. Added "free labs" as clear CTA
6. Added "your dreams just got a lot closer" framing from Schumer
7. Added 25 years experience mention
8. Added scoring system details (Curiosity Index, etc.)
9. Added small business owner as explicit audience

### Round 2 — Factual Accuracy Check

1. **FIXED: Family AI Bot description in landing page** was inaccurate ("answers questions, manages schedules"). Corrected to match syllabus: "Record your weekly family meeting. AI transcribes it, extracts tasks, books calendar events, creates a meal plan, and generates a shopping list."

2. **FIXED: Social media Tweet #3** presented a hypothetical as fact ("A small business owner replaced three subscriptions"). Changed to "Imagine replacing..." framing since we don't have real testimonials yet.

3. **FIXED: Missing "Agentic Mindset" concept** — added dedicated section to main document explaining the prompt->evaluate->improve->repeat cycle.

4. **FIXED: Missing "soft and hard skills" mention** — added to Applied AI section of main document.

### Round 3 — Principle Alignment Check

All documents now pass against all applicable GWTH principles. Some principles (like video walkthroughs, scoring metrics) don't apply to every document type — marketing copy doesn't need to detail the video walkthrough process, for example.

### Round 4 — Tone and Honesty Check

- No unsubstantiated claims found
- All statistics are sourced (Anthropic Economic Index, OpenAI, BCG, METR)
- No fear-mongering without empowerment follow-up
- No promises about specific outcomes — language uses "will learn" and "will build" (verifiable) not "will earn $X" (speculative)
- The "no discount codes" policy is explained honestly in every document that mentions pricing
- Independence claim is supported throughout

### Round 5 — Cross-Document Consistency Check

- Pricing is consistent across all documents ($37.50/mo x3, $7.50/mo ongoing)
- Course structure is consistent (Month 1: 24 lessons, Month 2: 20+15, Month 3: 20+15)
- Capstone descriptions are now consistent with syllabi
- "No coding" message is consistent and prominent in all docs
- Free labs CTA appears in all customer-facing documents
- Tech Radar "47+ tools" mention is consistent
- "25 years" experience claim is consistent where used
- Dynamic scoring description is consistent

## Remaining Considerations (Not Fixes, But Notes)

1. **Social proof gap**: All documents would benefit from real student testimonials once available. Currently using data points and logical arguments instead.

2. **Team pricing**: Mentioned as "coming soon" in employer doc. Should be finalised before that page goes live.

3. **Age appropriateness**: Main doc mentions teenagers. May want to clarify minimum age (16?) when the course launches.

4. **Video walkthrough**: The principle states each lesson has a project section "we then walk through with a video." This is an important feature but is a course delivery detail, not marketing copy — it's appropriately omitted from most marketing materials. Could be mentioned in a "How It Works" section.

5. **The Agentic Mindset principle** is only explicitly mentioned in the main document. Consider weaving this concept into the landing page and email sequence as well, since it's a key differentiator from other AI courses.
